 Дайте ответы на след. вопросы: 
- Что такое Linux и зачем она нужна. 
- Чем отличается CLI от GUI. - Что такое SSH. 
- Как сгенерировать пару RSA-ключей и зачем это делать. 
- Как скопировать открытый ключ для пользователя USER на сервер SERVER. 
- Какие задачи решаются с помощью команды 'gzip'. 
- Как перенаправить вывод какой-либо команды в файл с целью сохранения. 


1. Linux это ОС, ядро кот. распространяется на бесплатной основе и эксплуатируется преимущественно на серверах. Создана Торвальдсом Линуксом 17.09.1991г. 
Ядро - главная программа. Ядро выступает посредником между устройствами (видеокартой, процессором, оперативной памятью и пр.) и его ПО.
2. CLI(command line interface) от GUI это пользовательские интерфейсы. Главное отличие в том что графический интерфейс пользователя GUI(graphical user interface) позволяет пользователю взаимодействовать с системой с помощью графических элементов (окна, значки, меню), а интерфейс командной строки CLI позволяет пользователю взаимодействовать с системой с помощью команд.
3. ssh (secure shell) защищенное соединение к удаленному сервису (одноименный протокол), зашифрованый канал связи с сервером.
4. RSA используется и для шифрования данных для защиты информации при ее передаче, и для создания цифровых подписей.
Асимметричное шифрование — это метод шифрования данных, т.е. использование двух ключей — открытого и закрытого. Открытый (публичный) ключ применяется для шифрования информации и может передаваться по незащищенным каналам. Закрытый (приватный) ключ применяется для расшифровки данных, зашифрованных открытым ключом. Открытый и закрытый ключи — это очень большие числа, связанные друг с другом определенной функцией, но так, что, зная одно, крайне сложно вычислить второе.В такой схеме перехват любых данных, передаваемых по незащищенным каналам, не имеет смысла, поскольку восстановить исходную информацию возможно только при помощи закрытого ключа, известного лишь получателю и не требующего передачи.
5. Скопировать открытый ключ для пользователя USER на сервер SERVER с помощью команды: ssh-copy-id xsp@telran.ab88.space
6. С помощью команды 'gzip' сжимаются файлы для уменьшения размера.
7. Перенаправляем вывод команды с целью сохранения df -h > ./result_tati.txt - запись (если файл существует его перезапись)
df -h >> ./result_tati.txt - дозапись (если файл существует его дозапись)
ssh-copy-id USER@SERVER.
