First level: Укажите след. команды:

для полной очистки индекса:
			git reset  - команда использовалась для удаления из очереди на коммит;
			git rm --cached <file>  - удалять файлы из рабочего каталога и индекса и только из индексаБ

для просмотра разницы между раб.каталогом и посл. коммитом:
			(1) раб/каталог с индексом (git diff)
			(2) индекс с последним коммитом (git diff --staged)
			(3) раб/каталог с последним индексом (git diff HEAD)


для создания новой ветки и одновременного переключения в на нее:
			git checkout -b login - создать нов. ветвь и сразу на нее переключиться;

для сравнения веток между собой:
			git diff;

для удаления локальной ветки:
			git checkout master - переключаемся на главную ветвь, 
			git branch -v  - смотрим какие есть ветки,
			git branch -d название ветки - удаляем ненужную ветку,
			git branch -D название ветки - удаляем ненужную ветку принудительно после сообщения с подсказкой:


для слияния веток:
			git branch
			git diff ..название ветки - смотрим на разницу между ветками
			git diff master..название ветки - смотрим какие изменения будут привнесены,
			git merge название ветки - делаем слияние,
			git branch -d название ветки - удаляем ненужную ветвь.	


для перезаписи посл/коммита:
			git commit --amend;


Коротко опишите причину возникновения конфликтов при слиянии и порядок действий для их решения.
			Конфликт во время слияния может произойти в двух отдельных точках — при запуске и во время процесса
			слияния.
			git merge --abort -отменяем слияние если взник конфликт;
			git status -s  . выводим состояние ашего каталога, кот показывает конфликтный файл;
			редактируем конфликтный файл с помощью редактора nano, vim, или code;
			git add (file) . добавляем в очередь на коммит;
			git commit -аm "комент"  - выполняем фиксацию правок; 
			git status -s - проверяем состояние каталога
			git checkout master - переключаемся на основную ветвь;
			git merge ...(другая ветвь) - привносим изменения из др ветви в основную (слияние);
			git branch -d ...(другая ветвь) - удаляем ненужную ветвь.
